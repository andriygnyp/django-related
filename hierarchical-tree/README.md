# Deployment

Install and activate virtual environment:

```bash
$ python3 -m venv env
$ . env/bin/activate
```

Install `Django` and `Django rest framework`:

```bash
$ pip install -r requirements.txt
```

Start development server:

```bash
$ cd rest
$ ./manage.py runserver
```
